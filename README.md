# 🛡️ NavSafe – Smart Safety-Based Navigation System

NavSafe is a web-based navigation application that prioritizes user safety over speed. Unlike traditional navigation systems, NavSafe suggests routes based on safety factors such as trusted locations, time of day, and emergency conditions.

---

## 🚀 Features

### 🗺️ Smart Route Mapping
- Find routes between any two locations
- Visual route display using interactive maps

### 🧠 Safety Score System
- Each route is assigned a dynamic safety score
- Color-coded routes:
  - 🟢 Green → Safer
  - 🔴 Red → Risky

### 👥 Trusted Location Integration
- Routes near trusted locations are considered safer
- Visual markers shown on the map

### 🚨 Emergency Mode
- Prioritizes accessible and active areas
- Boosts safety score during emergencies

### 🌙 Night Safety Logic
- Detects night time automatically
- Adjusts safety score based on risk levels

### 📊 Real-Time Safety Info Panel
- Displays route safety status
- Shows context-based messages instead of alerts

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Leaflet.js (Maps)  
- OpenStreetMap  
- OSRM Routing API  

---

## 💡 How It Works

1. User enters start and destination
2. System fetches route using OSRM API
3. Safety score is calculated based on:
   - Distance to trusted locations
   - Emergency mode status
   - Time of day
4. Route is displayed with color coding and safety feedback

---

## 🎯 Key Innovation

NavSafe introduces **context-aware routing**, where navigation adapts based on user conditions like emergency situations and environmental factors, rather than only focusing on shortest path.
